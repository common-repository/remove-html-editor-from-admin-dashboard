This plugin will remove the HTML editor from the admin dashboard

== Description ==

This plugin removes the HTML editor from admin dashboard for restricted access to the theme files.

== Installation ==

1. Upload this plugin directory to `/wp-content/plugins/` 
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Editor Menu Item will no longer be available
